## react-webpack-boilerplate

基于 webpack4 快速搭建 react 开发环境，搭建教程[参考](https://github.com/KayneWang/blog/blob/master/webpack4-react.md)

### 执行命令

```shell
$ yarn install
```

### 本地开发

```shell
$ npm run start
```

打开 localhost:1234 查看效果

### 打包

```shell
$ npm run build
```

### 静态资源访问

```shell
$ npm run http-server
```
